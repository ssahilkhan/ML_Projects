# ML Projects

A collection of applied machine learning projects demonstrating real-world use cases across computer vision, time series forecasting, and natural language processing.

## Progress

![Progress](https://progress-bar.dev/50)

## Tech Stack

- **Python** - Core programming language
- **OpenCV** - Computer vision (face detection, gesture detection)
- **scikit-learn** / **TensorFlow** - ML frameworks
- **pandas** / **numpy** - Data manipulation
- **LSTM / RNN** - Time series forecasting (stock prediction)
- **NLP** - Spam detection using text classification

## Projects

- **Face Detection** - Real-time face detection using computer vision
- **Gesture Detection** - Hand/body gesture recognition
- **Stock Prediction** - Time series forecasting with deep learning
- **Spam Detection** - Text classification for spam filtering
